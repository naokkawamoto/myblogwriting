# check as LLM（レビュー専用プロンプト）

**方針**

- **特定のファウンデーションモデル（FM）に依存しない**。ローカルや任意のチャットで、このフォルダのテキストを指示として貼り、返答だけ受け取る運用を想定する。
- **プロンプトの正本は本リポジトリ**で管理する。評価側にモデル名を書かない（ベンダー中立）。
- 返答形式は各ファイルの「出力形式」に従う。**点数（数値）**が必要なものは、ファイル内で必須と明記する。

## ファイル

| ファイル | 用途 |
|----------|------|
| `evaluator_scored_generic.md` | 任意のテキスト成果物の採点つき批評（軸はカスタム可） |

SNS下書き専用の批評は `../social-company-pipeline/02_external_evaluator.md`（こちらも FM 非依存で使う）。

## 運用メモ

- 執筆用 LLM と **別セッション／別プロバイダ** にすると、甘い相互評価を避けやすい。
- 採点軸を変えたい場合は `evaluator_scored_generic.md` の「評価軸」セクションだけ編集する。
