# メタ用：画像生成プロンプトを1本で出力させる

**使い方**: このファイル全文を任意の LLM に貼り、下の **「用途モード」** と **「今回の内容」** を埋めて送る。  
出力は **画像生成AIにそのまま貼れる1本のプロンプト** とする。**特定のモデル名は出力に書かない**。

---

## 用途モード（依頼者が1つ選ぶ）

- **A — Web ヒーロー／OGP 向けインフォグラフィック（汎用）**  
  正本: `spec_hero_ogp_infographic.md`  
  （フラット＋軽いアイソメ、図解・短い日本語テキスト可。手順の細密さより **コンパクトな一枚** に向く。）

- **B — note 長文向け・編集リアル／シネマティック**  
  正本: `spec_note_editorial_cinematic.md`  
  （**写真／写実イラスト**、煽り広告にしない。**画像内の読めるテキストは原則なし**。）

- **C — note 記事アイキャッチ向け・横長 DX ワークフロー（既定）**  
  正本: `spec_note_dx_workflow_infographic.md`  
  （**プロ品質の青白インフォ**、ヘッダー／**左→右 5 ステップ**／**下部 3 ベネフィット**／締めコピー。**日本語キャプションを画像に載せる**。建設テック事例の note でよく使われる構成。）

依頼時に **A / B / C** を明記する。**note の事例・サービス説明のアイキャッチで未記入なら C** とみなす。
