# 改稿ルール

- フィードバックの **すべて** に従う必要はない。**ソースに反する指示は拒否**し、拒否した項目を末尾の「変更サマリー」に記す。
- 新しい数値・固有名詞・実績は **ソースにない限り追加しない**。
- トーンは初稿より **煽りを減らす方向**を優先する。

# 出力形式

1. **改稿版：X**
2. **改稿版：LinkedIn**
3. **変更サマリー**（箇条書き：採用したフィードバック／見送った点＋理由）
4. **人間レビュー用：残リスク**（最大5項目。例：事実確認が必要な一文、法的に敏感な表現 等）

以上を出力してください。
